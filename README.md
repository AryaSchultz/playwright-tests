# Playwright - SauceDemo

Projeto de automação de testes desenvolvido utilizando Playwright e TypeScript.

## Objetivo

Demonstrar boas práticas de automação de testes, organização do projeto e documentação de cenários utilizando Gherkin.

## Tecnologias

- Playwright
- TypeScript
- Node.js
- Dotenv

## Estrutura do Projeto

tests/
├── login/

Cada funcionalidade possui seus próprios cenários de teste, organizados de forma independente para facilitar a manutenção e escalabilidade do projeto.

## Variáveis de Ambiente

Embora o SauceDemo utilize credenciais públicas de demonstração, este projeto utiliza variáveis de ambiente para seguir boas práticas adotadas em aplicações reais, evitando que credenciais sensíveis sejam versionadas.
